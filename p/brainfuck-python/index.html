准备

Brainfuck是一种极小化的计算机语言，它是由Urban Müller在1993年创建的。由于fuck在英语中是脏话，这种语言有时被称为brainf*ck或brainf**k，甚至被简称为BF

Brainfuck的8种代码：

字符 含义
> 指针加一
< 指针减一
+ 指针指向的字节的值加一
- 指针指向的字节的值减一
. 输出指针指向的单元内容（ASCⅡ码）
, 输入内容到指针指向的单元（ASCⅡ码）
[ 如果指针指向的单元值为零，向后跳转到对应的]指令的次一指令处
] 如果指针指向的单元值不为零，向前跳转到对应的[指令的次一指令处

实现

我的基本思路是 用Python列表来模拟纸带 首先来创建一个函数

def paper_tape_machine(code):

我们要写一个函数对八种代码进行读取并进行操作，当然这不是仅进行一次的，在while循环和递归这两种方式中我选择了前者 绝对不是因为我在尝试后发现Python有最大递归深度 如下所示

def paper_tape_machine(code):
    pointer = 0
    paper_tape = [0]
    position = 0
    while position != len(code):
        position += 1

分别创建纸带列表 指针 位置（执行代码的位置） 变量

当position等于代码的长度时，即代码执行完毕，退出循环

然后加入对代码的判断 首先是>和<

指针移动，使其值加减1即可

我没有提前规定纸带列表的长度 所以我在使指针右移前会判断一下是否需要增加纸带长度：

 if code[position] == '>':
    if pointer + 2 > len(paper_tape):
        paper_tape.append(0)
    pointer += 1

左移时也需要进行一定的判断。当指针已经出于纸带第一位时，则无法继续往前，此时报错结束函数：

elif code[position] == '<':
            if pointer != 0:
                pointer -= 1
            else:
                print('Data overflow!')
                return

+和-只需要加减列表中单个元素的值 只需要注意其特殊规则：

255+1=0 0-1=255

elif code[position] == '+':
            if paper_tape[pointer] != 255:
                paper_tape[pointer] += 1
            else:
                paper_tape[pointer] = 0

elif code[position] == '-':
            if paper_tape[pointer] != 0:
                paper_tape[pointer] -= 1
            else:
                paper_tape[pointer] = 255

.和,需要将整数通过ASSII编码与字符进行转化，Python中已有函数chr() ord()能实现 直接使用即可

需要注意的是 输入时要检测一下字符串的长度

elif code[position] == '.':

elif code[position] == ',':

最后的结果是这样的

def paper_tape_machine(code,mode,strict):
    pointer = 0
    paper_tape = [0]
    position = 0
    i = 0
    while position != len(code):
        if code[position] == '>':
            if pointer + 2 > len(paper_tape):
                paper_tape.append(0)
            pointer += 1
        elif code[position] == '<':
            if pointer != 0:
                pointer -= 1
            else:
                print('Data overflow!')
                return
        elif code[position] == '+':
            if paper_tape[pointer] != 255:
                paper_tape[pointer] += 1
            else:
                paper_tape[pointer] = 0
        elif code[position] == '-':
            if paper_tape[pointer] != 0:
                paper_tape[pointer] -= 1
            else:
                paper_tape[pointer] = 255
        elif code[position] == '.':
            print(chr(paper_tape[pointer]),end='')
        elif code[position] == ',':
            text = input()
            if len(text) == 1:
                paper_tape[pointer] = ord(text)
            else:
                print('Invalid input!')
                return
        elif code[position] == '[':
            if paper_tape[pointer] == 0:
                position += 1
                while code[position] != ']' or i != 0:
                    if code[position] == '[':
                        i += 1
                    elif code[position] == ']':
                        i -= 1
                    if mode == 'debug':
                        print(position,pointer,code[position],i)
                    if position != len(code):
                        position += 1
                    else:
                        print('Syntax error! (Cannot find corresponding "]")')
                        return
        elif code[position] == ']':
            if paper_tape[pointer] != 0:
                position -= 1
                while code[position] != '[' or i != 0:  
                    if code[position] == '[':
                        i -= 1
                    elif code[position] == ']':
                        i += 1
                    if mode == 'debug':
                        print(position,pointer,code[position],i)
                    if position != 0:
                        position -= 1
                    else:
                        print('Syntax error! (Cannot find corresponding "[")')
                        return 
        else:
            if strict:
                print('Invalid code "',code[position],'" !')
                return
        if mode == 'debug':
            print(paper_tape,pointer,code[position],i)
        elif mode != 'normal':
            print('Invalid parameter')
            return
        position += 1
paper_tape_machine(input(),'normal',False)
